## Open New Tab in Firefox Browser after a time in a range given

This Script helps you to open a particular url in new tab of `Firefox` after randome time in given time range in seconds.

This can come handy in the use cases while you want to check one url repeatedly after after randome time in given time range in seconds.

`usage open_new.py <URL> <START_TIME_IN_SECONDS> <END_TIME_IN_SECONDS>`
